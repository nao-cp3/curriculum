●IT
  IT
    インフォメーションテクノロジー。インターネットなどの通信とコンピューターとを駆使する情報技術
  ICT
    インフォメーションアンドコミュニケーションテクノロジー。通信技術を使って人とインターネット、人と人が繋がる技術。
  IoT
    インターネットオブシングス。人を使わずモノが自動的にインターネットと繋がる技術のこと。

●OS
  OS
    オペレーティングシステム。操作・運用・運転を司るシステムソフトウェア。
  Windows
    Microsoftが提供しているOS
  macOS(Mac OS X)
    Appleが提供するMac用のOS
  UNIX
    コンピューター用のマルチタスク・マルチユーザーのOSの一種。安全で軽く安い。
  Linux
    OSの一種。オープンソースで無料。

●言語・環境
  マークアップ／フロントエンド
    HTML
      静的にWebページを作るためのマークアップ言語。要素の意味や情報構造を定義する。
    CSS
      HTML装飾するための言語。
    JavaScript
      プロトタイプベースのオブジェクト指向スクリプト言語。Webページに動きをつける。
  サーバーサイド
    .NET（C#, VB）
      ネットワーク上でアプリケーションを構築する基盤システムのこと。
    Java
      処理速度が速く、プラットフォームに依存しないオブジェクト指向の言語
      Java SE
        Javaの基本機能をまとめたもの
      Java EE（Jakarta EE）
        JavaSEを元にしてサーバーサイドの機能を追加したもの。
    Perl
      テキスト処理が得意なスクリプト言語。
    PHP
      動的にWebページを生成することができるサーバーサイドのスクリプト言語。
    Ruby
      オブジェクト指向スクリプト言語。

●サーバーソフト
  Apache
    Webサーバーソフトウェア。HTTPサーバー。
  nginx
    Webサーバーソフトウェア。高速・大量処理
  IIS
    Windows OSで動き、視覚的に操作可能

●SQL
  SQL
    データベース言語。データベースに対してしか命令することができない。
    DB2
      ミッションクリティカルなシステムや大規模システムで採用されることが多いリレーショナルデータベース製品。
    MySQL
      大容量でも高速で動作するデータベース管理システム。
      Maria DB
        リレーショナルデータベースとしてデータにアクセスするためのSQLインターフェースと高度なRDBMS機能の提供。
    Oracle DB
      関係データベース管理システム。二次元の表形式でデータを著しし、SQL言語によって検索・入力・消去の処理ができる。
    PostgreSQL
      リレーショナルデータベース管理システム。作成・編集・データの読み書きができる。
    SQLite
      データベース操作の他、トランザクション処理やトリガー、ビュー機能にも対応してる。
    SQL Server
      利用者の認証や権限管理、暗号化などセキュリティ機能が充実してる
  CRUD
    データを取り扱うソフトウェアに要求される4つの機能。作成・読み出し・更新・削除。
  トランザクション
    互いに関連・依存する複数の処理をまとめ一体不可分の処理単位として扱う。
    コミット
      処理や変更などを確定させる、反映させる。
    ロールバック
    稼働していたある状態に戻して復旧を試みること。
  排他制御（排他ロック）
    データベースシステムで記憶領域への同時アクセスを制限するロック機能の一つ。
    楽観ロック
    悲観ロック
  実行計画
  チューニング
    情報システムやコンピューター、ソフトウェアの設定や構成を調整したりする作業。
  パフォーマンス
    機器やソフトウェア、システムなどの処理性能や実行速度、通信回線・ネットワークなどの伝送速度や容量のこと。
  スロークエリ
  クエリログ
    データの問合せや要求などを一定の形式で文字に表したもの。
  IDE（統合開発環境）
    ソフトウェア開発に必要なソフトウェアを一つに組み合わせ、同じ操作画面から統一的な操作法で利用できるようにしたソフトウェアパッケージ。
    Eclipse
      Javaを始めとするプログラミング言語による統合開発環境。
    NetBeans
      Java、HTML5、PHP、C/C++の統合開発環境
    SpringToolSuite
      Java Springの開発環境
  DBツール
    A5:SQL Mk-2
    pgAdmin
    DBeaver
    MySQL Workbench
    Sequel Pro
    TeamSQL

  SE システムエンジニア
    情報システムの企画、設計、開発、試験、構築、導入、運用、更新、修正、廃棄などに携わる技術者の総称。狭義には、ソフトウェアの開発に携わる技術者のうち、プログラミング以外の業務を担当する者のこと。
  デザイナー
  エンジニア
    フロントエンド・エンジニア
    サーバーサイド・エンジニア

●UI（ユーザーインターフェース）
  UI
    ユーザーインターフェースとは、機器やソフトウェア、システムなどとその利用者の間で情報をやり取りする仕組み。
    CUI
      文字による表示や入力を基本とするCUI（Character User Interface：キャラクタユーザインターフェース）
    GUI
      絵や図形による表示、画面内の位置を指示することによる操作を基本とするGUI（Graphical User Interface：グラフィカルユーザインターフェース）

●ファイル/ディレクトリ
  ファイル
  拡張子
    ファイルの種類や形式を表すためのモノ
  ディレクトリ
    ルート・ディレクトリ
      装置やシステム全体の最上位のディレクトリのこと。
    カレント・ディレクトリ
      現在のディレクトリ
  パス
    絶対パス
      最上位ディレクトリから目的のディレクトリやまでの道筋を省略なくすべて記述する方法
    相対パス
      指し示したいディレクトリやファイルの相対位置を記述する方法

●ネットワーク
  LAN
    限られた範囲内にあるコンピュータや通信機器、情報機器などをケーブルや無線電波などで接続し、相互にデータ通信できるようにしたネットワークのこと。
  WAN
    通信事業者が設置・運用する回線網のことを指す
  クライアント
    他のコンピュータやソフトウェアから機能や情報の提供を受けるコンピュータやソフトウェアのこと
  サーバー
    他のコンピュータに対し、自身の持っている機能やサービス、データなどを提供するコンピュータのこと。

●各種プロトコル
  HTTP
    WebサーバとWebクライアントの間でデータの送受信を行うために用いられるプロトコル（通信規約）。
  FTP

  HTTPS
    通信方式の種別などを表すURIスキームの一つで、Webのデータ転送に用いられるHTTPが、SSLやTLSで暗号化されている状態を表したもの。
  SSL
    インターネットなどのIPネットワークでデータを暗号化して送受信するプロトコル（通信手順）の一つ。
  SSH
  ポート、ポート番号
    同じコンピュータ内で動作する複数のソフトウェアのどれが通信するかを指定するための番号。
  HTTP、FTP、SSHの各ポート番号
  ファイアウォール
  TCP
    インターネットなどのネットワークで、IP（Internet Protocol）の一段階上位層のプロトコル（通信規約）として標準的に使われるものの一つ。
